oacd_dummy
==========

oacd_dummy is a dummy media plugin for [OpenACD](https://github.com/OpenACD/OpenACD). It is written in [Erlang](http://erlang.org) and it uses [rebar](https://github.com/basho/rebar) as its build tool.
